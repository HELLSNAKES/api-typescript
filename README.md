# Api-typescript

## Setup

#### 1. Clone the repository.
```
git clone https://github.com/HELLSNAKES/api-typescript.git
```
#### 2. Navigate to the directory.
```
cd api-typescript
```
#### 3. Install the dependencies.
```
yarn install 
```
#### 4. Start the application.
```
npm run build:start
```
#### 5. Open http://127.0.1.1:5000
